### 公用样式名及其对应用途
| 公用样式名 | 用途 |
| ------------ | ------------ |
|Default|屏幕下方台词|
|Eng|专有名词英文标注|
|Narration|顶部台词或解说|
|Note|顶部注释|
|OP|OP中文歌词|
|ED|ED日语歌词及振假名|
|ED2|ED中文歌词|
|IN|插曲歌词|
|Title|各话中文标题|
|Staff|字幕组Credits表记|
|Screen|屏幕字（初始样式与Default一致）|
