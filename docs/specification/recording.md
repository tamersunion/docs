# 原文听写

## 内容要求

1. 单独语气词不用字幕  如：「哈哈哈哈」「嘿嘿嘿嘿」「啊啊啊啊」。

2. 注意不要遗漏上集回顾与下集预告。

3. 注意区分不送气浊音、不送气清音、送气清音之区别。

4. 台词翻译中通常不使用逗号、顿号等标点符号，结束部分不加句号，问号叹号等除非特别需要一般也不使用。

## 注释
听写之中如果有不懂或者不确定的地方，一般用 **\*** 或 **#** 标注。
> 特别注意：请不要使用全角字符（例：**＊** **＃** ）。

例：你好 *****

需要注释的地方，请使用 **/\*** 和 **\*/** 来包围。

例：你好  **/\*** 这里不确定是否正确 **\*/**
