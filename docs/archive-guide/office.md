# Office文档查看与编辑

档案库使用ONLYOFFICE组件实现在线预览与编辑功能。单击doc、docx、xls、xlsx、ppt、pptx等文档即可在线预览和写作编辑文档。

详细使用说明请在编辑器窗口，点击左上角 **文件** > **帮助** 查阅使用说明。
