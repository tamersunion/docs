# 登录

# 直接登录

登录档案库所使用的账号密码 与主页一致 。

# 统一认证登录

由于系统更换，SSO系统尚未完成对接，暂时关闭统一认证登录功能。

# 忘记密码

若忘记密码，可点击 忘记密码 按钮找回密码，该按钮将会引导你重置主页密码（档案库密码与主页一致，重置主页密码的同时也会重置档案库密码）。
