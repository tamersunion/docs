# 文本查看与编辑

# 使用方法

档案库内置文本编辑器，可用于编辑纯文本文件，单击文本文件即可打开编辑器。

# 保存

编辑器在线实时保存，编辑完毕后点击X即可关闭编辑器。

由于本组使用多文件版本管理体制，因此如果要修改稿件，请先复制一份新的再编辑。

# 特别提示

档案库自带的编辑器功能较为简单，推荐在本地编辑完成后再上传到档案库。
