# 文件分享

# 创建共享

对于非公共文件夹中的文件文件夹（即不放在公共文件夹里的文件/文件夹），点击该文件的分享按钮，在右侧弹出框内点击共享选项卡，创建共享链接，即可创建公开共享链接。

# 编辑共享

创建链接后点击共享链接旁边的...图标，可以设置允许编辑、隐藏下载（只能在线观看）、密码保护、设置过期日期、设置备注、以及取消共享等功能；同时还可以创建一个新的共享链接，用于设置不同的分享规则。

点击 取消共享 后该链接将立即失效。
